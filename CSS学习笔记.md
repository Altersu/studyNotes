# CSS学习笔记

## 1. 选择器的介绍

```text
1.标签选择器:使用的是Html的标签名
2.通配符选择器:该选择器可以与任何元素匹配.{例如:	*{color:red}}
3.类选择器:使用的是Html标签中的class属性(使用的方法是.类名),也可以使用标签名.类选择器确定某一个标签的类选择器.
4.id选择器:使用的是Html标签中的id属性(使用的方法是#id名),也可以使用标签名#类选择器来确定某一个标签的id选择器.
5.组合选择器:使用,(逗号)进行连接两个或多个标签.
6.
*id选择器和类选择器的不同,一般情况下id是唯一的,并且javascript也会用到id属性
这三个选择器的优先级别:标签<类选择器<id选择器
```
